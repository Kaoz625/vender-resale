# Launch Checklist — Vender Resale

## Step 1: Legal Entity
- [ ] Register LLC or Corp with NY Secretary of State (dos.ny.gov)
- [ ] Get EIN from IRS (irs.gov — free, takes 5 minutes online)
- [ ] Open business bank account

## Step 2: Tax & Licensing
- [ ] Apply for NY Sales Tax Certificate of Authority (tax.ny.gov)
- [ ] Get Resale Certificate so you can buy wholesale tax-free
- [ ] Get Workers' Compensation Insurance — Form C-105.2 (private carrier) or U-26.3 (State Insurance Fund)

## Step 3: Compliance
- [ ] Read Directive 4911 in full: https://doccs.ny.gov/system/files/documents/2024/11/4911_0.pdf
- [ ] Read Directive 4911A: https://www.ny.gov/sites/default/files/2023-04/4911a_Packages.pdf
- [ ] Confirm you are NOT on the Disapproved Vendor List: https://doccs.ny.gov/system/files/documents/2026/03/incarcerated-individual-disapproved-package-vendor-list-3-10-2026.pdf
- [ ] Download facility operations manuals for target facilities from doccs.ny.gov/legal

## Step 4: Sourcing / Suppliers
- [ ] Find wholesale food suppliers (Restaurant Depot, Sysco, US Foods, RFS)
- [ ] Find wholesale clothing supplier (blank sweatsuits, thermals, socks — all must be DOCCS-compliant)
- [ ] Find wholesale hygiene supplier (alcohol-free products only)
- [ ] Verify all products meet Directive 4911 specs before ordering inventory

## Step 5: Operations
- [ ] Set up shipping account with UPS or FedEx (USPS also works)
- [ ] Set up order management system (Shopify is simplest for catalog + checkout)
- [ ] Build product catalog — list facility-approved items only
- [ ] Create packaging workflow — weigh and track every package against 40 lb limit

## Step 6: Website & Marketing
- [ ] Build storefront at nyctailblazers.com/vender or sub-domain
- [ ] Set up vender@nyctailblazers.com email (Cloudflare email routing → catchall)
- [ ] Create catalog PDF for families
- [ ] Set up inmate search (facility + DIN lookup) so families can order by inmate ID

## Step 7: DOCCS Relationship
- [ ] Contact DOCCS Contract Procurement Unit: 518-457-8126
- [ ] Sign up for NY State Contract Reporter for bid notifications
- [ ] Consider applying for formal DOCCS procurement contracts (bigger volume)

## Step 8: Revenue
Target margin: 20-40% above wholesale cost
- Food: buy wholesale, sell at ~25% markup
- Clothing: buy wholesale, sell at ~40% markup  
- Hygiene: buy wholesale, sell at ~30% markup
- Bundles: pre-curated packages at slight premium
